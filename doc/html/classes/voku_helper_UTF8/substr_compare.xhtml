voku\helper\UTF8::substr_compare

Binary safe comparison of two strings from an offset, up to length characters.

    Signature

    public function substr_compare(string $main_str, string $str, int $offset, [integer $length = 2147483647, [boolean $case_insensitivity = false]] )

    Parameters

    $main_str — string
    The main string being compared.
    $str — string
    The secondary string being compared.
    $offset — int
    The start position for the comparison. If negative, it starts counting from the
    end of the string.
    $length — integer
    The length of the comparison. The default value is the largest of the length of
    the str compared to the length of main_str less the offset.
    $case_insensitivity — boolean
    If case_insensitivity is TRUE, comparison is case insensitive.
